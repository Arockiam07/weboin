# Weboin - 

Weboin is a premium digital agency website crafting intuitive UI/UX, scalable web applications, and strategic branding. This project is built with a focus on modern aesthetics, smooth animations, and high performance.

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript / JavaScript

## 🛠️ Setup Instructions

Follow these steps to get the project running locally:

### 1. Prerequisites
Ensure you have **Node.js** (v18.x or later) and **npm** installed on your machine.

### 2. Installation
Clone the repository and install the dependencies:
```bash
npm install
```

### 3. Running the Development Server
Start the development server with:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 4. Building for Production
To create an optimized production build, run:
```bash
npm run build
```
Then, you can start the production server:
```bash
npm run start
```

## ✨ Features & Assumptions

### Key Features
- **Modern UI/UX:** A sleek, dark-themed design tailored for a design agency.
- **Interactive Hero Section:** Engaging landing experience using Framer Motion animations.
- **Service Showcases:** Detailed presentation of agency offerings.
- **Portfolio Gallery:** Professional display of past work and case studies.
- **Responsive Navigation:** Smooth mobile-friendly navigation.
- **Contact Integration:** Ready-to-use contact section for lead generation.

### Assumptions
- The project assumes a modern browser environment for CSS4 and Framer Motion compatibility.
- Standard Node.js environment variables (if any) should be handled via `.env.local`.
- Assets like images and icons are served from the `public/` directory or external optimized sources.
